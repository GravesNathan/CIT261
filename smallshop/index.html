Small Shop Demo

This page utilizes basic JavaScript functions and Objects to simulate a simple game shop. A users current cart can be stored using local storage api to hold for later. An empty cart button is built in to clear the contents of the cart. I also plan to provide a method of saving and retrieving a cart via JSON.

I'm working on adding Ajax to either update pricing or to provide a user item description. I could also use it to update the most "popular" purchase (though it's only based on local storage without a database). A chat box for "support" would be fun.
I'll probably include video, images, and audio descriptions later.
