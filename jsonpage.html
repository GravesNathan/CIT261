Json


What is Json?

Json is a language often used to store data as it is transmitted between servers. This is because most languages include a method of taking data, changing it into a json string, and then taking the json string and changing it back into the original data type. This allows servers to communicate across different computer lanugages and platforms by simply using Json in between. Also, note that JSON can store strings, numbers, objects, arrays, boolean, and null values. It cannot be used for functions, dates, and when a value is undefined.

Json format

A Json string begins and ends with braces. Inside the braces are name:value pairs separated by commas. The name of each "name":value pair must be enclosed in quotation marks. The same is true of string values. An example Json string would be as follows.

{"name":"John", "age": 50, "height": "6 feet", "hometown:Hillsbarrow"}

Json can also contain arrays by using brackets to enclose the array as part of a value pair.

{"name":"John", "age": 50, "height": "6 feet", "hometown":"Hillsbarrow", "children":["Jacob","Anabell","Bob","Kat"] }

We can even create an Json string that contains an array which contains objects

{"name":"John", "age": 50, "height": "6 feet", "hometown":"Hillsbarrow","children":[
  {"name":"Jacob","age":15},
  {"name":"Anabell","age":14},
  {"name":"Bob","age":10},
  {"name":"Kat","age":4}
] }

Reading Json data

As you may have noticed the Json format coincides with that of the object format in Javascript. Likewise, we can reference the information contained in a json string much like we would in javascript. (See example below)

<script>
var myObj, x;
myObj = { name: "John", age: 30, city: "New York" };
x = myObj.name;
document.getElementById("demo").innerHTML = x;
</script>
Example taken from https://www.w3schools.com/js/tryit.asp?filename=tryjson_object_access, 10-11-2018

Note that we could also have used x = myObj["name"];

Json Parse

Parsing Json takes a Json string and changes it into an object. This is done with the JSON.parse method.

<script>
var txt = '{"name":"John", "age":30, "city":"New York"}'
var obj = JSON.parse(txt);
document.getElementById("demo").innerHTML = obj.name + ", " + obj.age;
<script>

Json Stringify

Taking an object (or other data type) and changing it to a Json format in preparation for transport is know as Stringifying. This is done witht he JSON.Stringify method.
