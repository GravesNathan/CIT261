JavaScript Basics

On this page I'll cover some common functions built into most modern programming lanuages. The topics I'll include are loops, conditioanl statements, functions, variables, parameters, arrays, and associative arrays. As these topics are widely available I'll provide a simple explanation and a JavaScript example of each. Credit for the examples goes to w3schools.com.

Loops

Loops are used to repeat code a certain number of times so long as a condition holds true. Different loop types include do...while, while, for, and for/in loops. The contnue function can be used to jump out of a current loop iteration and begin at the top of the loop. Break is used to terminate a switch or loop.

For Loop Example:
var i;
for (i = 0; i < cars.length; i++) {
text += cars[i] + "<br>";
}
//warning that i++ is executed before each iteration in Javascript. To do it after each iteration we can leave the third statment blank and include i++; at the end of the loop.

For/in Loop Example:
var person = {fname:"John", lname:"Doe", age:25};
var text = "";
var x;
for (x in person) {
text += person[x];
}
//The for/in loop is used to loop through each of the properties of an object

While Loop Example:
while (i < 10) {
text += "The number is " + i;
i++;
}
//the while loop will execute so long as the test statement is true. The statement is tested before the first iteration.

Do-While Loop Example:
do {
text += "The number is " + i;
i++;
}
while (i < 10);
//The do-while loop will execute at least once before the statment at the end is tested. It will continue to repeat so long as the statement is true at the end of each loop.

Conditional statements

The two most popular conditional statements consist of if...else statements and the switch statement. These are used to make decisions within a program based on current conditions. An if...else statement is usually preferred for small choices. A switch statement is often used when many choices are avaialable to make code easier to read and use.

If example
if (hour < 18) {
greeting = "Good day";
}

If Else example
if (hour < 18) {
   greeting = "Good day";
} else {
   greeting = "Good evening";
}

If Else If example
if (time < 10) {
   greeting = "Good morning";
} else if (time < 20) {
   greeting = "Good day";
} else {
   greeting = "Good evening";
}

Switch example
switch (new Date().getDay()) {
case 0:
   day = "Sunday";
   break;
case 1:
   day = "Monday";
   break;
case 2:
   day = "Tuesday";
   break;
case 3:
   day = "Wednesday";
   break;
case 4:
   day = "Thursday";
   break;
case 5:
   day = "Friday";
   break;
case 6:
   day = "Saturday";
default:    day = "Something went wrong. Today doesn't have a defined name.";
}
 Note that the "default" keyboard is used to specify what happens if none of the previous cases matched our test statement.

Parameters and Functions

Functions are a set of instructions that can be carried out whenever the function is called. This helps reduce code redunancy by allowing a single body of code to be executed from anywhere on the page. A return statement is used to exit a function. It can also be used to return data from within the function to the place it was originally called.

Functions can have parameters in their function statements. These are placeholder variables that can be used to store information that is passed into the function when it is called. If a function is called with less parameters than those that are available in the functions head then the remaining parameters are set to undefined for the function interation.

Simple function
function myFunction(a, b) {
  return a * b;
}

Function call (or invoked)
myFunction(10,2);

fullName is an object method (covered in objects page). The full name function is called using the .call method, but is used on the person2 object.
var person = {
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
}
var person1 = {
   firstName:"John",
   lastName: "Doe",
}
var person2 = {
  firstName:"Mary",
  lastName: "Doe", }
person.fullName.call(person2); // Will return "Mary Doe"

The apply() method can also be used. It is similar to the call() function, but takes arguments as arrays. The call() method takes arguments idividually.

Variables

Variables differ in javascript in that there is no assigned datatype durin declaration. Instead the language will choose a data type based on the information that is stored. Variables can be created as follows.
var x, y, z; x = 5; y = 6; z = x + y;

Arrays

Arrays are a type of object which is used to contain a list. Array entries are assigned numeric values beginning with the value of 0 and incremented by 1. The index is used when refering to specific items in the list.

Array Syntax:
var array_name = [item1, item2, ...];

Example:
var cars = ["Saab", "Volvo", "BMW"];

You can also use the JavaScript "new" keyword to create an new JavaScript Array type object.

Example
var cars = new Array("Saab", "Volvo", "BMW");
We can also declare an empty array and then add elements to it.

Associative Arrays

It is important to note that JavaScript doesn't provide support for associative arrays. If we try to create an associative array JavaScript will instead interpret it as an object. This will also produce confusion because we cannot use .length or other standar array notations properly. If you're looking for a key:value pairing it is better to simply create a proper object which is covered in the object section.

Bad "Associative" Array example (Don't use)
var person = [];
person["firstName"] = "John";
person["lastName"] = "Doe";
person["age"] = 46;
var x = person.length; // person.length will return 0
var y = person[0]; // person[0] will return undefined

Note that person["firstName"] will work, but creating an object properly is still recommended.
